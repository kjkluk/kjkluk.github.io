普通物理学公式集--力学类

力学

运动学

  • 位移: \(\Delta x\)
  • 速度: \(v = \frac{dx}{dt}\)
  • 加速度: \(a = \frac{dv}{dt}\)
  • 等加速度运动:
    • \(v = v_0 + at\)
    • \(x = v_0t + \frac{1}{2}at^2\)
    • \(v^2 = v_0^2 + 2ax\)
  • 圆周运动:
    • 角速度: \(\omega = \frac{d\theta}{dt}\)
    • 向心加速度: \(a = \frac{v^2}{r} = r\omega^2\)

动力学

  • 运动定律: \(F = ma\) (牛顿第二定律)
  • 动量: \(p = mv\)
  • 冲量: \(I = \int F dt\)
  • 动量守恒定律: \(\sum p_i = \sum p_f\)
  • 功: \(W = \int F \cdot dr\)
  • 动能: \(K = \frac{1}{2}mv^2\)
  • 势能: \(U\) (取决于力场)
  • 机械能守恒定律: \(E = K + U = 常数\)
  • 功率: \(P = \frac{dW}{dt} = F \cdot v\)

振动和波动

  • 单摆振动: \(x = A\cos(\omega t + \varphi)\)
  • 单摆的周期: \(T = 2\pi\sqrt{\frac{l}{g}}\)
  • 波的速度: \(v = \lambda f\)
  • 波的公式: \(y = A\cos(\omega t - kx + \varphi)\)

热力学

  • 热力学第一定律: \(\Delta U = Q - W\)
  • 理想气体状态方程: \(PV = nRT\)
  • 卡诺循环效率: \(\eta = 1 - \frac{T_c}{T_h}\)

电磁学

电磁学

  • 库仑定律: \(F = k\frac{q_1q_2}{r^2}\)
  • 电场: \(E = \frac{F}{q}\)
  • 电势: \(\varphi = \frac{U}{q}\)
  • 静电容量: \(C = \frac{Q}{U}\)

磁学

  • 磁场: \(B\)
  • 安培定律: \(F = BIL\sin\theta\)
  • 洛伦兹力: \(f = qvB\sin\theta\)
  • 法拉第电磁感应定律: \(\mathcal{E} = -\frac{d\Phi}{dt}\)
  • 电感: \(L\)

光学

  • 折射定律: \(n_1\sin\theta_1 = n_2\sin\theta_2\)
  • 透镜公式: \(\frac{1}{f} = \frac{1}{u} + \frac{1}{v}\)

现代物理学

  • 光电效应方程: \(E_k = h\nu - W\)
  • 德布罗意波长: \(\lambda = \frac{h}{p}\)
  • 不确定性原理: \(\Delta x \Delta p \ge \frac{\hbar}{2}\)
  • 质能等价: \(E = mc^2\)
